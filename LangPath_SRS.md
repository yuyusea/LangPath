# Software Requirements Specification (SRS) & System Testing

## Project Information

**Project Title:** LangPath - AI 기반 외국어 학습 컨설턴트 서비스

**Project Description:**  
외국어 학습자들은 자신에게 맞는 학습 계획을 세우는 것에 어려움을 겪으며, 체계적인 학습 관리 없이 동기를 잃기 쉽다. 본 프로젝트는 AI 기반 맞춤형 학습 스케줄 생성과 진도 관리 기능을 제공하여 학습자가 목표를 달성할 수 있도록 돕는 개인 맞춤형 학습 컨설턴트 플랫폼이다. 7단계 채팅 형식의 온보딩으로 학습자 프로필을 수집하고, OpenAI를 활용해 12주간의 상세한 학습 로드맵을 자동 생성한다. 일일 대시보드, 과제 체크리스트, 연속 학습 일수 추적, 주간 캘린더 등을 통해 학습 동기를 유지하고 꾸준한 학습 습관을 형성할 수 있도록 지원한다.
