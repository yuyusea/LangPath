AI는 어느 부분에 들어가



AI가 들어가는 부분을 정리해줄게.

AI 활용 지점
1️⃣ 스케줄 생성 (핵심!)
사용자가 채팅으로 답변한 내용을 AI에게 전달:


"사용자 정보:
- 언어: 일본어
- 현재 수준: 히라가나 읽을 수 있음
- 목표: 3개월 내 일상회화
- 하루 학습 시간: 1시간
- 선호 방식: 회화 중심
- 약점: 듣기

위 정보를 바탕으로 12주 학습 스케줄을 만들어줘.
각 주차별 목표와 일일 학습 내용을 구체적으로."
AI가 출력:

월간 로드맵 (1개월차, 2개월차, 3개월차)
주간 스케줄 (월~일 각각 무엇을 할지)
각 학습 항목의 구체적인 내용
2️⃣ 초기 컨설팅 챗봇 (선택사항)
정해진 7가지 질문만 할 거면 AI 없이도 가능하지만,
AI를 쓰면 더 자연스러운 대화 가능:


사용자: "일본어 공부하고 싶은데 어디서부터 시작해야 할지 모르겠어"

AI: "좋아요! 몇 가지만 물어볼게요. 
     히라가나나 가타카나는 읽을 수 있나요?"
3️⃣ 스케줄 수정/조정

사용자: "이번 주는 바빠서 30분밖에 못해"

AI가 스케줄 재조정:
"알겠어요! 이번 주 스케줄을 30분 버전으로 줄여드릴게요.
 다음 주에 조금 더 보완하는 걸로 할까요?"
4️⃣ 학습 자료 추천 (선택사항)

오늘 학습 주제: "일본어 기본 인사말"

AI가 추천:
- 유튜브 영상 3개
- 추천 앱 (Duolingo, Anki)
- 연습 문제 5개 생성
5️⃣ 주간 리포트 생성 (선택사항)

사용자의 이번 주 학습 데이터를 AI에게 전달:
- 완료한 날: 월, 화, 목, 금 (4일)
- 건너뛴 날: 수, 토, 일
- 학습한 내용: 문법 3개, 단어 80개

AI가 생성:
"이번 주 정말 잘하셨어요! 💪
 4일 연속 학습하셨네요. 
 특히 단어 암기가 탁월했어요!
 
 다만 듣기 연습이 좀 부족했는데,
 다음 주는 드라마 보기를 추가해볼까요?"
최소 버전 (AI 꼭 필요한 부분)
시작 단계에서는 1번 스케줄 생성만 AI 써도 충분해.


[사용자 7가지 질문 답변 완료]
      ↓
[AI에게 프롬프트 전송]
      ↓
[AI가 12주 스케줄 생성]
      ↓
[화면에 표시]
나머지는 나중에 추가해도 돼!

구체적인 AI 사용 예시
AI에게 보낼 프롬프트 예시:


당신은 외국어 학습 컨설턴트입니다.
아래 학습자 정보를 바탕으로 맞춤 학습 스케줄을 만들어주세요.

[학습자 정보]
- 학습 언어: 일본어
- 현재 수준: 히라가나 읽기 가능
- 목표: 일상 회화 (JLPT N4 수준)
- 기간: 3개월
- 일일 학습 시간: 1시간
- 선호: 회화 중심
- 약점: 듣기

[요구사항]
1. 12주 주차별 학습 목표
2. 각 주의 월~금 일일 학습 내용 (구체적으로)
3. 각 학습 내용은 1시간 내 완료 가능하게
4. 회화와 듣기에 더 많은 시간 할당
